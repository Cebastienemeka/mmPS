MM Pilates Studio static website

Files included:
- index.html
- about.html
- contact.html
- css/style.css
- js/main.js
- assets/images/

How to run locally:
1. Open the folder in VS Code or any editor.
2. Start a local server in this folder, for example:
   python3 -m http.server 8000
3. Open http://localhost:8000/index.html

Notes:
- Bootstrap and Bootstrap Icons are loaded from CDN.
- The contact form is a front-end demo and shows a success message only.
- You can connect the form to Formspree, Netlify Forms, or your own backend.
